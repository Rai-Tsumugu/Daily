# Daily

日常的な活動を24時間記録するwebアプリケーションです。運動・睡眠・学習などの粒度で記録できます。このサービスによって、自分が過去に何をしていたのかを計り分析し、振り返ることができるようになります。

## 概要

このプロジェクトは、[失われていく些細な日常の活動を記録することで、過去の自己分析を容易にすること]を達成するために作成されました。
主な機能として、以下のような特徴があります。

- **活動の記録**
  - 24時間の活動を連続的に記録（前の活動の終了時間が次の開始時間と一致）。
  - 睡眠・学習などのタグ付け、モチベーションの多段階管理、詳細な内容保存。
  - 活動中の割り込み入力機能（例：学習中に「休憩10分」を挿入するだけで時間を自動調整）。
- **データ分析**
  - 活動時間・累積回数の自動計算。
  - 数年分のデータを対象とした、目標値や中央値との差分の算出。
  - 累積時間や回数の推移をグラフで可視化。
- **振り返り**
  - 目標や中央値との差分を確認。
  - 日・週・月単位での振り返りコメント入力。
